## ID System (Hierarchy)
- **$SID** (Story Project): `$OBSIDIAN_01`
- **@WID** (World): `@VIRELITH_501`
- **#CID** (Character): `#ELARA_902`

Hierarchy: `$SID` > `@WID` > `#CID`

---

## 5-Phase Development
1. **Foundation** (0-20%): Name, Role, Genre.
2. **Personality** (20-40%): Motivations, Flaws, Shadow.
3. **Backstory** (40-60%): Ghost, Transforming Events.
4. **Relationships** (60-80%): Allies, Enemies, Dynamics.
5. **Arc** (80-100%): Growth, Theme, Climax.
