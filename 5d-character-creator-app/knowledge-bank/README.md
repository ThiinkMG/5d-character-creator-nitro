# Knowledge Bank

This folder contains reference materials and PDFs used for research and development.

## ⚠️ Important Note

**Large PDF files are excluded from Git** due to GitHub's 100 MB file size limit. These files are stored locally for reference purposes only.

## Files

The following reference materials are available locally (not tracked in git):

- Psychology and Psychiatry references
- Storytelling and writing guides
- Character development resources

## Usage

These files are for local reference only. If you need access to specific materials, they should be obtained separately or stored using Git LFS if needed for the project.
